## Project Overview
This project builds a complete end-to-end data analytics pipeline to analyze retail sales,
customer behavior, and inventory performance using multiple tools.

## Tools & Technologies
- Excel (Data Cleaning & Preparation)
- SQL (Data Storage & Querying)
- Python (EDA & Customer Segmentation)
- Power BI (Dashboard & Reporting)

## Business Objectives
- Analyze sales performance across stores and staff
- Identify high-value customers
- Perform customer segmentation using RFM analysis
- Monitor inventory and stock levels
- Provide management with interactive dashboards

## Project Workflow
1. Data cleaning and preprocessing using Excel
2. Relational database creation and querying using SQL
3. Exploratory Data Analysis and RFM segmentation using Python
4. Interactive dashboard creation using Power BI

## Key Insights
- Identified high-value and at-risk customer segments
- Certain stores and staff contribute significantly to total revenue
- Inventory alerts highlight low-stock products

## Conclusion
This project demonstrates an industry-level analytics workflow integrating Excel, SQL,
Python, and Power BI to deliver actionable business insights.
